Краткое описание преимуществ системы multibooker

Multibooker легко устанавливается

Для запуска системы необходмио:

  1. Заключить договор с театром.
  2. Обеспечить в театре интернет.
  3. Установить Google Chrome на всех компьютерах, где будет использоваться multibooker.
  4. Открыть порт 8080.

Далее мы у себя на сервере создаем копию системы и даем доступ театру.

Система работает по адресу theatre_example.multibooker.ru

Всё!

Возможности

Multibooker позволяет эффективно и удобно вести билетное хозяйство, в системе предусмотрено все необходимое, от редактора схем залов, до гибкой настройки продаж.

Все необходимое для работы с системой описано в подробной документации.

Отчетность

Каждая операция, проводимая в системе, логируется в отчетности, таким образом можно в любой момент получить актуальную информацию по любому мероприятию / кассиру / агенту / квоте / билету / заказу и т.д.

API

API multibooker позволяет выводить всю необходимую информацию в любую внешнюю систему, например мы можем транслировать актуальную информацию по любому "срезу" БД в мобильное приложение. Таким образом, обладатель приложения будет всегда в курсе всех событий по всем подведомстевнным театрам.

СКД

в Multibooker предусмотрена система контроля доступа, которая позволяет в реальном времени отслеживать проходы на мероприятие.

Если в театре уже установлено оборудование СКД, то мы просто с ним интегрируемся, в противном случае, мы готовы предоставить быстрое решение со сканерами штрих-кодов, с которыми multibooker уже синтегрирован. Если по каким-либо причинам, сканеры не подходят, театр самостоятельно выбирает оборудование и устанавливет его, после чего мы с ним интегрируемся.

Продажа через интернет

Любое мероприятие можно легко продавать через интернет, для этого необходимо сделать пару настроек и вставить небольшой кусок кода на сайт, с которого будут идти продажи.

Электронный билет

Пример электронного билета, который приходит покупателю на почту.

Подключение музеев:

Выставка в музее отличается от мероприятия в театре, отсутствием точной даты, времени и указания места в зале.

В мультибукере предусмотрен механизм создания и продажи входных билетов без указания мест.

Дополнительно возможно расширить функционал возможностью настройки диапазона дат выставки, возможности ограничения продаж билетов по времени, настройки тарификации для разных категорий посетителей.
