Краткая инструкция

Как быстро запустить продажу онлайн билетов на различные мероприятия с помощью системы multibooker.

Простота работы с системой

Быстрый запуск работы системы осуществляется на нашем оборудовании через облачные решения.

Чтобы приступить к продажам билетов нужно осуществить нескольков шагов, описанных далее.

Создание схемы зала

С помощью редактора можно создать любую схему вашего зала.

Создание фондов зала

Это объединение нескольких мест в группы для удобства разграничения прав доступа к местам.
Например, места можно вынести в служебный фонд, недоступный для продажи в кассах.

Определение расценки фондов

Для расценки групп мест используется схема распоясовки.

Ценовые группы предназначены для возможности объединить в группу места с одинаковой стоимостью.

При создании распоясовки мероприятия на схеме зала появятся все пояса из заведенных справочников.

Создание мероприятия

Можно создать несколько вариантов мероприятий:

1. Мероприятия с входными билетами без указания мест.

2. Мероприятия с указанием мест на билетах с привязкой к схеме зала.

Создание входных билетов на мероприятие

Можно создавать любое количество типов входных билетов:
- взрослый,
- детский,
- автомобили,
- танцпортер,
- студенческий.

Указывается название типа билета, цена и количество.

Продажа через интернет

Настроить созданные мероприятия для продажи их через интернет.

Настройка фрейма интернет продаж

Для продажи через интернет используется фрейм, который открывается на вашем сайте.

Для этого в настройках требуется указать пары мероприятие - фрейм, заполнить параметры продажи, наценка, сервисный сбор и пр.

Внедрение виджета на сайт

Чтобы фрейм работал, нужно вставить небольшой html-код на ваш сайт.

Фрейм поддерживает все браузеры (internet explorer 9+).
