Краткий мануал о том, как быстро запустить продажу онлайн билетов на паромы с помощью системы multibooker.

Создать мероприятие без схемы зала

Создание мероприятия без указания схемы зала подразумевает мероприятие со входыми билетами.

Создать входные билеты на мероприятие

Можно создавать любое количество типов входных билетов, взрослые, детские, автомобили и т.д.

Указывается название типа билета, цена и количество.

Сгенерировать такие же мероприятия на каждый день при помощи модуля генерации репертуара

Проставить сгенерированным мероприятиям настройки, позволяющие продавать их через интернет

Настроить фреймы продаж

Указать пары мероприятие - фрейм, заполнить параметры продажи, наценка, сервисный сбор и пр.

Внедрить виджет на свой сайт
