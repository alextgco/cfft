CRM пользователи

Описание модуля «CRM пользователи» в системе multibooker

CRM модуль

Модуль позволяет объединять данные пользователей в единую базу как с интернет заказов, так и с касс.

На данный момент в таблице предусмотрены поля:
- ФИО,
- e-mail,
- телефон,
- данные банковской карты.

Таблица расширяема, и мы легко добавим любые поля для вашей базы данных.

Таблица фильтруется по любому полю.

Фильтры легко настраиваются и позволяют оперативно найти любого пользователя. Также имеется быстрый поиск по ключевым полям.

Анкета пользователя в заказах

Окно оформления заказа имеет анкету пользователя и данные о выбранных билетах.

Для оформления заказа вы можете выбрать уже имеющегося crm пользователя или оформить нового.

Анкета имеет следующие поля:
- crm клиент,
- телефон клиента,
- e-mail клиента.

Количество полей с параметрами анкеты настраивается и может быть расширено по вашему желанию.

Фрейм - поля платёжной формы

Выбранные поля с параметрами для анкеты пользователя можно выводить и на платёжной странице.

Количество и параметры полей также могут быть расширены по вашему желанию.

Рассылка

Можно осуществлять рассылку по любым «срезам» базы crm пользователей.

Возможно настроить рассылку как через e-mail, так и через sms.

Наличие окна вёрстки шаблонов письма для рассылки.

Визуализация статистики

Вся информация из базы может визуализироваться в виде графиков и диаграмм с возможностью гибкой настройки и фильтрации.

Интеграция
